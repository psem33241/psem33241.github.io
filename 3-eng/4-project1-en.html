FR-CARBURANT : Python, SQL & BI Project

August 2024 - September 2024

Welcome to our innovative project to create an interactive dashboard for comparing diesel prices in France.

This project, undertaken before my formal training, represents a personal challenge that I set myself to evaluate my initial skills and familiarize myself with the practical aspects of the Data Analyst profession.

Context

Living in Toulouse's Place du Capitole and owning a diesel vehicle, I needed an efficient tool to quickly access diesel prices at nearby service stations. The aim was to target the cheapest stations within an adjustable radius, in order to make substantial savings.

Data

The data came from data.gouv.fr and were processed using Python (Pandas, Geocoding), then stored in a MySQL database. The results were visualized using Power BI, enabling precise, real-time monitoring of diesel prices.

Dashboard

Our interactive dashboard offers the following features:

  • View diesel prices at all service stations in France, with a geographical focus on Toulouse.
  • Calculate the distance between fuel stations and my address using Haversine's formula.
  • Quick identification of the cheapest stations within a defined, adjustable radius.
  • Automated price updates and real-time display.
  • Display average prices when several stations are selected.

The impact

Although this dashboard is initially designed for personal use, it can also be used by transport companies to optimize their fuel costs, generating significant savings.

Areas for improvement

For a future version of the project, improvements include :

  • Use actual position to calculate distances more accurately.
  • Integrate road distances for greater precision.
  • Enhance user interface for greater intuitiveness.
